Getting Started with the Web

This site is a practice ground for our class. You’ll see how HTML structures content, how CSS styles it, and how JavaScript adds interactivity. We’ll start simple and build confidence through short, focused exercises.

What You’ll Learn Today

By the end of this lesson, you’ll be able to identify block and inline elements, navigate the DOM tree using your browser’s developer tools, and create a small page with a header, navigation, content, and a footer.

You’ll also practice using attributes like src, href, and alt, and you’ll submit a simple form to understand how data is grouped and sent from the browser.

Technology in Everyday Life

Technology shapes how we communicate, learn, and create. From quick video calls to online classrooms, the modern web makes powerful tools available to anyone with a browser. Building a solid foundation in HTML helps you understand how those tools are assembled.

Why Structure Matters

Clean, semantic structure improves accessibility for screen readers, boosts search visibility, and makes collaboration easier. Think of HTML as the blueprint that everyone on your team can read.

Next Steps

After today’s session, try converting a plain text document into a web page using headings, paragraphs, links, and images. Then, replace generic <div> containers with semantic elements like <header>, <section>, and <article> where it makes sense.
